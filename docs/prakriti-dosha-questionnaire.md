## Scoring Logic

**Value Mapping:**
- Option A = Vata Points
- Option B = Pitta Points
- Option C = Kapha Points

**Calculation:** Sum the total for each column across all 18 questions.

**Example:** Vata (10), Pitta (5), Kapha (3)

**Output:**
- **Primary Dosha:** Highest score (e.g., Vata)
- **Bi-Doshic:** If second highest is within 15–20% of the first (e.g., Vata-Pitta)
- **Tri-Doshic:** If all three scores are nearly equal (rare)
